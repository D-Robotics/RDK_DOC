
### Installation

```
$ npm install

```

### Local Development

默认构建中文手册：
```
$ npm run start
```

仅构建英文手册：

```
$ npm run build  -- --locale en
```


### Build

默认构建

```
$ npm run build
```


### Deployment


```
$ npm run serve
```

默认`IP`打开网址：
中文手册：http://localhost:3000/docs/
英文手册：http://localhost:3000/en/docs/
